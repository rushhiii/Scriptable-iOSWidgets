## What this collection is

This repository contains hand-crafted Scriptable widgets for iOS. Each widget is a standalone script that you drop into the Scriptable app, configure once, and add to your home screen.

- Consistent setup across every widget.
- Clear parameters for fast customization.
- Offline-friendly behavior where possible.

## Quick start

1. Install Scriptable on your iPhone or iPad.
2. Pick a widget from the list below.
3. Copy the JavaScript file into Scriptable.
4. Add a Scriptable widget and select the script.
5. Adjust parameters to match your layout.

## Design notes

<Callout title="Keep it lean" variant="note">
  These widgets are optimized for fast refreshes. If you add new APIs, keep the payload small and cache results whenever possible.
</Callout>

## Need help

If a widget fails to load, run the script directly in Scriptable first. Most issues come from missing API keys, incorrect sheet URLs, or location permissions.
