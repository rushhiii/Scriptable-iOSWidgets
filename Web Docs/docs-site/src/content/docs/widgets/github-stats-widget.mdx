## Overview

A comprehensive GitHub widget with profile stats, repo metrics, and contribution heatmaps. It supports caching for offline use and more than twenty themes.

## Setup

1. Create a GitHub personal access token with read permissions.
2. Store the token in Scriptable using Keychain.set.
3. Update the username in the script.
4. Add a Scriptable widget and select the script.

## Parameters

- auto, light, dark, blue, night, indigo: theme presets.
- heatmap,theme: show the contribution heatmap.
- username/repo,stars,theme: repo-specific stats.

## Notes

<Callout title="Token safety" variant="warning">
  Do not share your GitHub token. Store it only in Scriptable Keychain.
</Callout>
