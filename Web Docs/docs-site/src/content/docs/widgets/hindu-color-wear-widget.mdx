## Overview

A daily color recommendation widget based on Hindu traditions. It automatically detects the day and adapts the layout for small, medium, and large widgets.

## Setup

1. Copy the script into Scriptable and save it.
2. Add a Scriptable widget and select the script.
3. No parameters are required.

## Sizes

- Small: shows the color and day.
- Medium: shows a weekly grid.
- Large: shows the daily reasoning.
