## Overview

A schedule widget that reads a public Google Sheet CSV and shows todays classes. It can also render a full week grid or simulate specific times.

## Setup

1. Publish a Google Sheet to the web as CSV.
2. Set SHEET_URL in the script.
3. Add a Scriptable widget and select the script.

## Parameters

- full view: weekly grid.
- test mon 10:30: simulate a time and day.
- get tue: show a specific day.

## Notes

The widget refreshes every 15 minutes. Use full view on large widgets for best readability.
