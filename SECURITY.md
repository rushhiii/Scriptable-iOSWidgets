# Security Policy

## Reporting a vulnerability

If you discover a security issue, please do not open a public issue.

Instead, report it privately by email:

- `rushiofficial1205@gmail.com`

Please include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Suggested fix (if available)

We will review reports as quickly as possible and coordinate a fix.

## Scope

This policy applies to repository code and scripts in this project.
