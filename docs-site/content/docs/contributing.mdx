## What to include

- A short description of the widget and its purpose.
- The Scriptable script file.
- A README-style setup section with parameters and data sources.
- At least one screenshot.

## Submitting updates

1. Fork the repository.
2. Create a feature branch.
3. Add or update the widget files.
4. Open a pull request with a clear summary and screenshots.

## Report issues

If something is broken, open a GitHub issue with:

- Widget name.
- Steps to reproduce.
- Screenshots of the error.
- Your iOS version.
