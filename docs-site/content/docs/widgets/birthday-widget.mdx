## Overview

A small widget that shows your age with two-decimal precision, total days lived, and a ring that tracks time to your next birthday.

## Setup

1. Copy the script into Scriptable and name it BirthdayWidget.
2. Add a small Scriptable widget and select the script.
3. Set the widget parameter as: name, Month Day Year.

Example parameter:

```
Rushi, May 11 2005
```

## Notes

- The widget is optimized for small size only.
- The theme uses a dark gradient background by default.
